Создаем на Github репозиторий cwp-16, клонируем его, открываем в IDE
С помощью библиотеки axios будем отправлять GET-запросы к Google Geocode API
Например, получим гео-информацию по Минску:
https://maps.googleapis.com/maps/api/geocode/json?address=Minsk
Создаем скрипт async.js. В этом файле будем использовать операторы async/await для ожидания промисов
1. С помощью Google Geocode API отправим параллельные запросы на информацию о городах - Минск, Мадрид, Рим. Из ответов выведем соответствия город - страна
2. С помощью Promise.any получим страну этих городов - Париж, Ницца
3. Получим информацию об улице Via Nicola Salvi и последовательно получим координаты для всех элементов из address_components
Установим билблиотеку co
Создаем скрипт co.js. В этом файле будем использовать библиотеку co и оператор yield
1. С помощью Google Geocode API отправим параллельные запросы на информацию о городах - Брест и Минск. С помощью geolib вычислим расстояние между ними
2. С помощью Google Geocode API и Promise.mapSeries отправим последовательные запросы на информацию о городах - Минск, Копенгаген, Осло, Брюссель. С помощью geolib.findNearest найдем ближаший к Минску город
3. Получим информацию об улице Piazza del Сolosseo и последовательно получим координаты для всех элементов из address_components
