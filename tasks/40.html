ЛР40
Создаем на Github репозиторий cwp-40, клонируем его, открываем в IDE
Копируем файлы из cwp-39
Устанавливаем react router
С помощью React Router добавим в приложение страницу с данными об авторе приложения, а также вынесем на отдельную страницу отображение итоговой информации о маршруте (количетсво точек, общее время в пути и задействованные виды транспорта)
При реализации будем использовать Browser Router, поэтому донастроим серверную часть
* Настроим взаимодейтсвие React Router с Redux Saga
* Если при загрузке приложения текущая страница "об авторе", то не будем загружать данные о маршруте до тех пор, пока пользователь не перейдёт на "маршрут" или "итоговая информация"
Задания со звездочкой предназначены для персонального роста и необязательны к показу при защите лабораторной работы
